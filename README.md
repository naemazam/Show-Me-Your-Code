## 📊 Project Evaluation Criteria
Projects will be evaluated based on the following:

| Criteria | Description |
| :--- | :--- |
| **Innovation** | How unique the idea is |
| **Technical Implementation** | Code quality & architecture |
| **Practical Value** | Real-world usability |
| **UI/UX** | User experience |
| **Documentation** | Clarity of project explanation |
